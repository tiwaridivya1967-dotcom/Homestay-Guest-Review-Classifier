# Homestay-Guest-Review-Classifier
The above web tool is about helping a homestay business to automatically understand customer reviews. The web application classifies guest reviews as positive, neutral or negative, identifies themes such as host, location and cleanliness. It provides AI generated responses and also suggests the ideas of improvement.

## Tech Stack
  -React.js
  -Node.js
  -Express.js
  -Gemini API
## Features
   -Sentiment Analysis( Positive, Negative, Neutral)
   -Theme Detection( Host, Location, Cleanliness, Value, Experience)
   -AI-generated Management Responses
   -AI-based Improvement Suggestions
   -Batch Review Analysis
